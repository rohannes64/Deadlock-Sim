Project Overview

This toolkit automatically detects potential deadlocks in system processes by analyzing process dependencies and resource allocation graphs. It identifies circular wait conditions and suggests resolution strategies using DFS cycle detection and the Banker's Algorithm.

Algorithms Implemented

1. DFS Cycle Detection — Builds a Resource Allocation Graph (RAG) and uses depth-first search to find back-edges which indicate circular waits (deadlocks).

2. Resource Allocation Matrix Method — Mark-sweep algorithm: iteratively marks processes that can complete given available resources, unmarked processes are deadlocked.

3. Banker's Algorithm — Safety check for resource allocation. Determines if a safe execution sequence exists given current allocation and maximum needs.
